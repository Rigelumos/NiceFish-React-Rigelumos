## 介绍

这是 NiceFish 的 React 版本，基于React 18.0.0 ，使用 Antd、Inversify、 定制版 Bootstrap开发。

## 用法

打开你的命令行终端，执行以下命令：
```
   git clone https://gitee.com/mumu-osc/NiceFish-React.git
   cd NiceFish-React
   yarn or npm i
   yarn start or npm start
```  

## 在线演示

- NiceFish-React 在 GitHub Pages 上的演示地址： https://yanyunchangfeng.github.io/NiceFish-React

## 主要依赖

- React 18.0.0
- Antd 4.19.5
- Bootstrap 4.2.1
- Echarts 4.2.1
- braft-editor 2.2.9
